Setup → Plugin Installation

In this step-by-step guide we will install the Plugins required to enable all the theme's functionality.

Install the Plugins Login to your WordPress admin panel, and navigate to REDMAG Tools → Install.

If you want to sure you will install like as demo , you should install and active all plugins

Select all the plugins in the list and choose Install from the Dropdown menu, then click Apply.

Then click to Return to required plugins installer

Then you will see this interface

That's it!

Besides you can install turn one by plugin , click Plugins then install plugins that you want , but plugins with required badge are essential to make REDMAG work smoothly.
