Customize Homepage → Short code : Silder

click to → Na Silder

1.Layout

This short code has 5 layouts and we used it on demo site

2.Type Post

This short code has 2 post type :News , Featured

News is will show the posts latest news

Featured is will show the posts Featured

How to show posts Featured ?

  1. Go to Post → All Post
  2. Tick on the star

So you can create 3 layout for Silder with Latest news or posts Featured
